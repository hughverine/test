# 為替レート取得ツール

## 概要
このアプリケーションは、為替レートをリアルタイムで取得・表示するツールです。

## 機能
- USD/JPYの為替レート表示
- 履歴データのグラフ表示
- その他の通貨ペアの表示

## 使用方法
1. アプリを起動します
2. 必要な通貨ペアを選択します
3. データが表示されます

## 技術スタック
- Python
- Streamlit
- Selenium
- Pandas
- Matplotlib

## デプロイ
このアプリはHugging Face Spacesでホストされています
